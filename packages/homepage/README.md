# Tool Factory Homepage

Landing page for the Tool Factory suite showcasing all available and upcoming factories.

## Development

```bash
npm install
npm run dev
```

Opens at http://localhost:3000

## Features

- **Factory Showcase**: Cards for each factory with descriptions
- **Status Badges**: "Available" vs "Coming Soon" indicators
- **Direct Navigation**: Click available factories to open them
- **Premium Design**: Dark theme with gradients and hover effects
- **Responsive**: Mobile-friendly layout

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide Icons
